# AI/ML Developer Portfolio

A premium, modern portfolio website for **N. Uthkarsh Sai**, an AI/ML developer. This project showcases technical expertise, featured projects (including AI and Engineering simulations), and contact information.

## 🚀 Features
- **Modern Design**: Dark aesthetic with amber accents, glassmorphism, and premium typography.
- **Dynamic Projects**: Showcases 8 distinct projects ranging from Deep Learning (RAG Bots, Study Assistants) to Engineering simulations (ANSYS Bridge Analysis).
- **Responsive Layout**: fully optimized for mobile, tablet, and desktop viewing.
- **Zero-JS Mobile Menu**: Efficient checkbox-based navigation for mobile devices.
- **Custom Assets**: High-quality, AI-generated project visuals and engineering simulation screenshots.

## 🛠️ Tech Stack
- **Structure**: HTML5
- **Styling**:  CSS3 (Grid, Flexbox, Custom Properties)
- **Icons**: Custom SVGs (Feather & System icons)
- **Fonts**: Google Fonts (Syne for headings, DM Mono for content)

## 📦 Setup Instructions

1. **Download the Project**:
   Ensure you have the main HTML file: `portfolio.html`.

2. **Organize Assets**:
   The portfolio references custom project images. For the best experience, ensure the images are placed in an `assets/` or `images/` folder and that the paths in `portfolio.html` are updated accordingly if you move them.
   - Current images for new projects are located in the local Gemini brain directory.

3. **Dependencies**:
   No external dependencies or build steps are required. All styling is self-contained in the HTML file, and fonts are loaded via Google Fonts CDN.

## 🏃 How to Run the Project

Since this is a client-side static site, you can run it directly:

### 1. Simple Open
Simply double-click `portfolio.html` to open it in your default web browser (Chrome, VS Code Live Server, Firefox, etc.).

### 2. Using Live Server (Recommended for Development)
If you are using **VS Code**:
1. Install the **Live Server** extension.
2. Right-click `portfolio.html`.
3. Select **"Open with Live Server"**.
This will provide a local development URL (usually `http://127.0.0.1:5500`) and automatically refresh when you save changes.

### 3. Deployment
The project is ready to be hosted on **GitHub Pages**, **Vercel**, or **Netlify**. Simply upload the `portfolio.html` file and any associated images.
