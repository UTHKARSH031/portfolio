## ⚙️ Deployment (GitHub Pages)

1. Go to the repository **Settings**
2. Navigate to **Pages**
3. Under **Build and deployment**:
   - Source → `Deploy from branch`
   - Branch → `main`
   - Folder → `/ (root)`
4. Click **Save**

Your site will be available at:

```
https://uthkarsh031.github.io/portfolio/
```
